# News Headlines Web Scraper

## Description
A simple Python web scraper that fetches the latest news headlines from Hacker News using Requests and BeautifulSoup.

## Features
- Fetches latest news headlines
- Saves headlines to a text file
- Uses BeautifulSoup for HTML parsing

## Technologies Used
- Python
- Requests
- BeautifulSoup4

## Installation

Install the required packages:

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
py scraper.py
```

## Output

The program creates a file named:

```
headlines.txt
```

containing the latest news headlines.
